## 📌 Overview
A first-person 3D maze exploration game inspired by the world-famous Wolfenstein 3D, built entirely from scratch in C using the Ray-Casting algorithm and the miniLibX graphics library.

## 🚀 Features
- **Ray-Casting:** Real-time 3D rendering from a 2D map.
- **Textures:** Wall texturing depending on orientation (North, South, East, West).
- **Movement:** Player movement (WASD) and camera rotation.
- **Map Parsing:** Strict `.cub` map file validation.

## 🛠️ Usage
```bash
make
./cub3D maps/map.cub
```
